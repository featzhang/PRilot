Setup Guide

1. Install GitHub CLI (gh)

This tool uses gh CLI to query GitHub API. Install it first:

macOS (Homebrew)
brew install gh
Ubuntu / Debian
sudo apt install gh
Windows (winget)
winget install GitHub.cli

More options: https://cli.github.com

2. Authenticate with GitHub

Login to your GitHub account:

gh auth login

Follow the interactive prompts. Choose HTTPS protocol and authenticate via browser.

Verify it works:

gh auth status

3. Install Node.js

Node.js >= 16 is required to run the server.

node --version  # check if installed

Install from https://nodejs.org or via nvm:

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts

4. Start PRilot

npm install
node server.js

Then open http://localhost:3000 in your browser.

5. Usage Tips

  • Enter a repository (e.g. apache/flink) and optionally an author filter
  • Click Search to load PR list
  • Refresh uses server cache; Force Refresh bypasses cache and queries GitHub directly
  • Auto Refresh can be toggled on/off with configurable interval
  • Click the CI / Azure CI button on a PR card to view CI pipeline status
